# Todo List 
    A simple todo using Postgres, Flask, and React
---
## Summary
--- 
### Requirements
- Postgres
- React
- Python3 

### Run Instructions
- In two separate terminals
#### Backend
     cd utils
     virtualenv venv
     source venv/bin/activate
     pip install -r requirements.txt
     python3 api.py
    
#### Frontend
    cd src
    npm install
    npm start
